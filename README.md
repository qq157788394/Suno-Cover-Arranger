# Suno Cover Arranger - Suno封面编曲器

一个基于React和Ant Design Pro开发的Suno封面编曲器应用，提供三栏独立滚动布局，用于生成和管理Suno音乐封面和歌词。

## 功能特点

### 🔹 高效的三栏操作界面
- **左侧栏**：轻松浏览和选择音乐风格，快速配置封面参数
- **中间栏**：便捷编辑和管理歌词，支持段落式排版
- **右侧栏**：实时预览生成效果，所见即所得
- 独立滚动设计，操作互不干扰，提升创作效率

### 🔹 便捷的资源入口
- 一键直达Suno官方邀请页面，获取创作权益
- 快速访问DeepSeek开放平台，探索更多AI创作可能性
- 简洁直观的导航，让资源获取更高效

### 🔹 专业的封面生成体验
- 丰富的预设音乐风格，满足多样化创作需求
- 智能歌词格式化，让歌词呈现更专业
- 实时预览生成结果，及时调整优化
- 一键生成功能，快速完成封面创作

### 🔹 流畅的用户体验
- 界面简洁现代，操作逻辑清晰
- 无页面级滚动，仅区域内局部滚动，专注创作内容
- 响应式设计，适配不同屏幕尺寸
- 优化的性能，流畅的交互体验

## 快速开始

### 环境准备

安装依赖：

```bash
npm install
```

### 启动项目

```bash
npm start
```

项目将在 http://localhost:8000 启动

### 构建项目

```bash
npm run build
```

### 代码检查

```bash
npm run lint
```

自动修复lint错误：

```bash
npm run lint:fix
```

## 项目结构

```
src/
```

## 核心功能说明

### 1. 样式选择
- 提供多种预设音乐风格
- 可自定义封面参数
- 实时预览样式效果

### 2. 歌词编辑
- 支持段落式歌词编辑
- 自动格式化歌词结构
- 实时预览歌词效果

### 3. 结果生成
- 一键生成完整封面
- 支持结果预览和复制
- 提供生成历史记录
